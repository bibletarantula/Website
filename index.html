Welcome to Project EPF!


Project EPF is an all new open source HTML5 CPPS client which will be made publicly available for use among developers within the CPPS community. Project EPF is designed to be easy to setup no matter what skill level you're at. If you have a basic understanding of C# and a VPS or a PC capable of hosting, you'll be on your way in no time!
